# Horror FPS Template — Лаба №2 по ОПД

Учебный проект по дисциплине ОПД. Система контроля версий в Unreal Engine 5.

## ⚠️ Content Warning
Проект содержит jumpscare-эффекты, резкие громкие звуки и тёмные локации.

## Технологии
- Unreal Engine 5.7.4
- Шаблон: Horror FPS Template (Fab)
- Git + Git LFS

## Как склонировать
```bash
# Установите Git LFS: https://git-lfs.com/
git lfs install
git clone https://github.com/UliaGrishkova/laba2_unreal.git
cd laba2_unreal
git lfs pull
```

## Как запустить
1. Установите Unreal Engine 5.7.4 через Epic Games Launcher
2. Двойной клик по `Horror_FPS_Template.uproject`
3. При запросе выберите версию 5.7
